# Pharmacy Textbook Library

A comprehensive collection of **92 pharmacy textbooks** sourced from the Internet Archive (public domain and open access).

## Contents by Category

| Category | Books | Size |
|----------|-------|------|
| Clinical Pharmacy | 21 | ~1.1 GB |
| Pharmacognosy | 20 | ~1.3 GB |
| Pharmacy Practice | 35 | ~1.3 GB |
| Pharmaceutics | 13 | ~434 MB |
| Pharmacology | 2 | ~89 MB |
| Toxicology | 1 | ~588 KB |

**Total: 92 books | ~4.1 GB**

## Source
All books were downloaded from [Internet Archive](https://archive.org) and are public domain or open access materials.

## Usage
This repository uses [Git LFS](https://git-lfs.github.com/) for PDF files. Make sure you have Git LFS installed before cloning:

```bash
git lfs install
git clone https://github.com/festus557/pharmacy-textbook-library.git
```

## Note
Some books are historical (1800s–early 1900s) and represent the evolution of pharmacy practice. Modern textbooks are also included.
